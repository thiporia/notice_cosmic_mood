# 개인정보처리방침 모음

## Structure

- `index.html`: Landing page.
- `privacy.html`: Privacy Policy for 감정 우주.
- `privacy-ledger.html`: Privacy Policy for 쉽게 쓰는 가계부.
- `css/`: Stylesheets.
- `js/`: JavaScript files.

## Deployment

This site is designed to be hosted on **GitHub Pages**.

1.  Push this repository to GitHub.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch as the source.
4.  The site will be live at `https://<username>.github.io/<repository-name>/`.
